# 📟 Tradutor de Código Morse

Este é um programa simples em Java que converte frases de texto para Código Morse. Foi o meu primeiro projeto para praticar lógica de programação e estruturas de dados.

## 🚀 O que este projeto faz
- Recebe uma frase do utilizador via terminal.
- Converte cada letra para o seu correspondente em Morse.
- Substitui espaços entre palavras pelo símbolo `|`.
- Ignora diferenças entre maiúsculas e minúsculas.

## 🧠 Conceitos de Java Aplicados
Neste projeto, aprendi e apliquei:
1. **HashMap** 🗄️: Para criar um dicionário de tradução eficiente (Chave: Letra, Valor: Morse).
2. **Scanner** ⌨️: Para ler a entrada de dados do utilizador.
3. **Laço For** 🔄: Para percorrer cada caractere da frase.
4. **Manipulação de Strings** 🧵: Uso de `charAt()`, `length()` e concatenação.
5. **Estruturas Condicionais** 🚦: `if/else` para tratar espaços e separadores.

## 🛠️ Como Executar
1. Certifica-te de que tens o Java instalado.
2. Compila o ficheiro:  
   `javac Main.java`
3. Executa o programa:  
   `java Main`
