# 📊 Descodificador de CSV

Este programa em Java lê ficheiros externos no formato CSV e organiza os dados no terminal. Foi desenvolvido para praticar a manipulação de ficheiros e o tratamento de fluxos de dados.

## 🚀 O que este projeto faz
- Lê automaticamente o ficheiro `dados.csv`.
- Separa nomes e idades usando a vírgula como delimitador.
- Trata erros de leitura com blocos de segurança.
- Limpa espaços desnecessários nos dados extraídos.

## 🧠 Conceitos de Java Aplicados
1. **File Reader & Buffer** 📂: Leitura de ficheiros externos.
2. **Try-Catch** 🛡️: Tratamento de exceções para segurança do código.
3. **Split & Array** 📑: Divisão e armazenamento de strings por separadores.
4. **While Loop** 🔄: Processamento de múltiplas linhas de dados.
5. **String Trim** 🧹: Refinação e limpeza de texto.

## 🛠️ Como Executar
1. Certifica-te de que o ficheiro `dados.csv` está na pasta correta.
2. Compila o ficheiro:
   `javac Main.java`
3. Executa o programa:
   `java Main`
